# Password Generator

A simple password generator application built using PyQt5.

## Description

The Password Generator is a desktop application that allows users to generate random passwords based on their preferred strength level and criteria. It provides options for selecting password strength, specifying the number of digits and special characters, and allowing users to start the password with a specific character.

## Features

- Generate random passwords based on user-selected criteria
- Options for selecting password strength (Weak, Strong, Very Strong)
- Specify the number of digits and special characters
- Start the password with a specific character
- Copy generated password to clipboard
- Simple and intuitive user interface

## Requirements

To run the Password Generator, you need to have the following dependencies installed:

- Python 3.x
- PyQt5 library
